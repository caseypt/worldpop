About the Data

Explore population data around the globe with the WorldPop Data Explorer. WorldPop is a premier open source high-resolution data source for population and demographic information.
WorldPop provides open access to spatial demographic datasets, covering aspects such as population counts, age structures, births, pregnancies and poverty.

All data is released under a CC-BY licensed and full available. Methodology can be found on WorldPop Methods page.
